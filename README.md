# PharmaCare — Pharmacy Landing Page Template

A clean, modern HTML landing page template for pharmacies and healthcare businesses. Cold white-green palette, medical canvas animations, zero external dependencies.

![Screenshot](preview.png)

## Demo
[Live preview](https://owlos.sk/templates/pharmacare/) *(update this URL after publishing)*

## Features
- Responsive layout (mobile-first, tested at 375px–1920px)
- Self-hosted fonts — GDPR-friendly, no Google Fonts CDN
- Zero external CDN dependencies (no Bootstrap, no jQuery, no icon fonts)
- Medical canvas animations: EKG heartbeat, molecule nodes, DNA helix, pulse rings
- Scroll-reveal animations via IntersectionObserver
- Animated stat counters
- Pharmaceutical cross loader
- localStorage-based demo banner dismissal
- Well-commented vanilla JS — no build step required

## Quick Start
1. Clone or download this repository
2. Open `index.html` in any modern browser — no build step required
3. Replace placeholder images in `assets/images/` with your own photos
4. Update branding (pharmacy name, phone, email, address) in `index.html`
5. Search for `TODO:` comments — they mark every line that needs your content
6. Remove the demo disclaimer banner block before going live

## Replacing placeholder content
Search `index.html` for `TODO:` — every placeholder is marked:
- Company name and registration number (footer)
- Phone number, email address, physical address (contact section)
- Opening hours and emergency contact
- Form handler URL (`action="#"`)
- Legal page links (Privacy Policy, Cookie Policy, Terms, Imprint)

## GDPR Notice
This template is designed with European privacy law in mind:
- All fonts are self-hosted — no requests to Google Fonts CDN
- No third-party CDN dependencies for CSS or JS
- No analytics, tracking pixels, or advertising scripts included
- Contact form is non-functional by default (no data collection)

If you add analytics or advertising scripts, you are responsible
for GDPR compliance (cookie consent, Privacy Policy) on your own website.

## Contact Form
The contact form is non-functional by default (`action="#"`).
To activate, replace `action="#"` with your backend endpoint or form service.
GDPR: if you collect personal data, you must add a Privacy Policy
and a consent notice near the form.

## Photos
Demo photos are **not included** in this template for redistribution.
Replace all images in `assets/images/` and any `src="depositphotos_*"` references
with your own licensed photos before publishing.
